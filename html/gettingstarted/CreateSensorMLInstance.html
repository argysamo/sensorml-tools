Create a SensorML Instance

Once you have a project opened in your workspace, select the project. Then select the SensorML Menu/New SensorML Instance. The "Save As" dialog will appear and require you to specify the project to which you are adding a SensorML Instance. The selected project will then appear as the "parent folder". At the bottom of the dialog, enter the name of your SensorML instance. For this tutorial, we will create a SensorML description of a Davis Weather station. Click "OK" to finish.
